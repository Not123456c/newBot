# ⚡ ابدأ بسرعة - Quick Start Guide

## 🚀 بدء سريع جداً (5 دقائق)

### الخطوة 1: التثبيت (1 دقيقة)
```bash
pip install -r requirements.txt
```

### الخطوة 2: الإعداد (2 دقيقة)
```bash
copy .env.example .env
# ثم عدّل BOT_TOKEN و SUPABASE في .env
```

### الخطوة 3: الاختبار (1 دقيقة)
```bash
python test_requirements.py
```

### الخطوة 4: التشغيل (1 دقيقة)
```bash
python final_bot_with_image.py
```

## 📞 ماذا لو حدثت مشكلة؟

### ❌ `ModuleNotFoundError`
```bash
pip install -r requirements.txt
```

### ❌ `Connection Error`
تأكد من:
- BOT_TOKEN صحيح
- SUPABASE_URL صحيح
- SUPABASE_KEY صحيح

### ❌ الصور غير واضحة
النظام يبحث عن الخطوط تلقائياً ✓

### ❌ PDF لا يعمل
ستحصل على نسخة نصية بدلاً منه ✓
