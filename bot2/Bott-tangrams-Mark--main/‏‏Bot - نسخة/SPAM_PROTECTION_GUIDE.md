# 🛡️ نظام الحماية من الطلبات المتتالية - دليل المستخدم

## 📖 نظرة عامة

تم إضافة نظام متقدم للحماية من الطلبات المتتالية والمزعجة التي قد تؤثر على أداء البوت. النظام يعمل تلقائياً ويرسل إشعارات للمسؤولين لاتخاذ القرار المناسب.

---

## 🚀 البدء السريع

### 1️⃣ إنشاء الجداول في Supabase

انسخ واستخدم الأوامر من ملف [SQL_SPAM_PROTECTION.md](SQL_SPAM_PROTECTION.md)

### 2️⃣ التحقق من التثبيت

عند بدء البوت، ستجد في logs:
```
✅ نظام الحماية من الطلبات المتتالية جاهز
```

### 3️⃣ الاستخدام العادي

البوت سيعمل تلقائياً دون أي إجراء من المستخدم. النظام:
- ✅ يراقب الطلبات من كل مستخدم
- ✅ يحدد الطلبات المريبة
- ✅ يرسل إشعارات للمسؤولين
- ✅ يسمح للمسؤول بالقرار

---

## ⚙️ الإعدادات

### تعديل حدود الطلبات

في ملف `spam_protection.py`، غيّر هذه القيم حسب احتياجك:

```python
self.MAX_REQUESTS_PER_MINUTE = 5      # أقصى 5 طلبات في دقيقة واحدة
self.MAX_REQUESTS_PER_5_MINUTES = 15  # أقصى 15 طلب في 5 دقائق
self.BAN_DURATION_MINUTES = 30        # مدة الحظر (30 دقيقة)
```

**مثال:**
- إذا أرسل مستخدم 6 طلبات في 30 ثانية → ⚠️ تحذير
- إذا أرسل 20 طلب في 4 دقائق → 🚨 إشعار للمسؤول

## 🔔 نظام الإشعارات

### كيف يعمل الإشعار:

عند اكتشاف طلبات مريبة:

1. **إرسال إشعار للمسؤول:**
```
🚨 تنبيه أمان: طلبات متتالية مريبة

👤 المستخدم: #12345678
📊 عدد الطلبات: `18` طلب في فترة قصيرة
⏰ الوقت: `14:32:15`
📅 التاريخ: `2024-03-19`

ماذا تريد أن تفعل؟
[✅ حظر المستخدم] [❌ تجاهل التنبيه]
```

2. **خيارات المسؤول:**
   - ✅ **حظر المستخدم**: يتم حظره فوراً لمدة 30 دقيقة
   - ❌ **تجاهل التنبيه**: تسجيل الحادثة كـ "مجاز" دون اتخاذ إجراء

3. **رسالة التأكيد:**
```
✅ تم حظر المستخدم #12345678 بنجاح
```

## 🔧 الوظائف الرئيسية

### الدوال المتاحة في `spam_protection.py`:

#### `is_user_blocked(user_id)`
التحقق من حالة الحظر:
```python
if spam_protection.is_user_blocked(12345678):
    print("المستخدم محظور")
```

#### `check_request(user_id)`
فحص طلب جديد:
```python
result = spam_protection.check_request(user_id)
# {
#     'is_spam': False,
#     'is_blocked': False,
#     'request_count': 3,
#     'max_allowed': 15
# }
```

#### `block_user(user_id, reason)`
حظر مستخدم يدويًا:
```python
spam_protection.block_user(12345678, "نشاط مريب")
```

#### `unblock_user(user_id)`
فك الحظر:
```python
spam_protection.unblock_user(12345678)
```

#### `log_spam_incident(user_id, request_count)`
تسجيل حادثة:
```python
spam_protection.log_spam_incident(12345678, 18)
```

```
الثانية 1: طلب 2
الثانية 2: طلب 3
الثانية 3: طلب 4
الثانية 4: طلب 5
```

**الإجراء:** 🚨 إشعار فوري للمسؤول


- المسؤول يختار: ❌ تجاهل
- ✅ يتم التسجيل دون حظر
