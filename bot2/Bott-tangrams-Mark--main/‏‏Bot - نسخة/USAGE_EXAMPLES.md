## مثال 2️⃣: الطالب يريد معرفة موقعه (هل هو من الأوائل 10)

### ما يفعله الطالب:
```
> يكتب: 10
> يضغط: إرسال
```

### النتيجة:
```
⏳ جاري حساب أفضل 10 طلاب... يرجى الانتظار.

[يظهر 10 طلاب بنفس الصيغة الأعلى]
```

### التفسير:
```
إذا رأى نفسه في القائمة:
✅ يعرف أنه من الأوائل 10

إذا لم يره:
ℹ️ قد يجرب 15 أو 20 للتعرف على موقعه الفعلي
```

## مثال 3️⃣: طلب غير صحيح

### ما يفعله الطالب:
```
> يكتب: 100 (أكتر من 50)
> يضغط: إرسال
```

### ما يظهر:
```
❌ الرقم يجب أن يكون بين 1 و 50. أرسل رقماً صحيحاً.
```

### شرح الخطأ:
```
- تم إدراك أن "100" هو طلب أوائل (أقل من 3 أرقام)
- لكن الحد الأقصى هو 50
- البوت يعطيك رسالة واضحة عن الحد الأقصى
```

## مثال 7️⃣: رسالة المساعدة

### ما يفعله الطالب:
```
> يكتب: /help
> يضغط: إرسال
```

### ما يظهر:
```
📖 مساعدة البوت

الأوامر المتاحة:
/start - البدء والقائمة الرئيسية
/help - هذه الرسالة
/admin - لوحة التحكم (للمدراء فقط)
/top - أفضل 10 طلاب

طريقة الاستخدام:
1️⃣ أرسل رقمك الجامعي مباشرة (مثال: 202201234)
2️⃣ أرسل اسمك لأبحث عنك
3️⃣ أرسل رقماً (3، 5، 10) لرؤية أفضل N طالب
   • أرسل 3 → أفضل 3 طلاب
   • أرسل 5 → أفضل 5 طلاب
   • أرسل 10 → أفضل 10 طلاب
4️⃣ استخدم /admin للأوائل بالمادة (مشرفين فقط)

📝 ملاحظة: يمكنك إرسال أي رقم من 1 إلى 50
```

## 📊 جدول السيناريوهات

| المدخل | النوع | النتيجة |
|-------|------|--------|
| `5` | طلب أوائل | أفضل 5 طلاب |
| `202201234` | رقم جامعي | نتائج الطالب |
| `أحمد` | بحث باسم | قائمة بالطلاب المطابقين |
| `/top` | أمر | أفضل 10 طلاب |
| `/help` | أمر | رسالة المساعدة |
| `/admin` | أمر | لوحة التحكم |
| `100` | طلب غير صحيح | خطأ (أكتر من 50) |
| `-5` | إدخال غير صحيح | البحث باسم (ليس رقم) |

## ⚡ نصائح سريعة

### نصيحة 1: الأرقام الأفضل للتجربة
```
3 → تجربة سريعة للأوائل
5 → الخمسة الأوائل الافتراضيين
10 → أوسع قليلاً
```

### نصيحة 2: للبحث عن نفسك
```
إذا أردت معرفة موقعك:
- جرب 10 أولاً
- إذا لم تكن فيهم جرب 20
- وهكذا حتى تجدك
```

### نصيحة 3: للمشرفين
```
استخدم /admin دائماً للإحصائيات المتقدمة
واستخدم 📚 الأوائل بالمادة لمقارنات معينة
```

## 🎯 حالات الاستخدام الشائعة

### الحالة 1: طالب يريد معرفة من الأفضل في الدفعة
```
الحل: أرسل 5 أو 10
```

### الحالة 2: معلم يريد أفضل الطلاب في مادة معينة
```
الحل: /admin → 📚 الأوائل بالمادة
```

### الحالة 3: طالب يريد معرفة موقعه بالضبط
```
الحل: جرب أرقام مختلفة حتى تجده (3، 5، 10، 15، 20)
```

### الحالة 4: إدارة تريد إحصائيات عامة
```
الحل: /admin → 📊 إحصائيات النظام
```
