---

|------|-------|----------|
| `/start` | البدء والقائمة الرئيسية | أول أمر |

## 📋 ملاحظات مهمة

### ⚠️ التحذيرات
- ❌ أوامر المشرفين محدودة على معرفات ADMIN_IDS
- ❌ `/upload` يدعم ملفات محددة فقط
- ❌ `/backup` يحتاج bucket معد مسبقاً

### ✅ الأفضليات
- ✅ جميع الأوامر الذكية للجميع
- ✅ أوامر المشرفين تحتاج ترخيص فقط
- ✅ رسائل الخطأ واضحة

## 🔄 دورة الاستخدام

```
المستخدم يأتي
    ↓
يرسل /help
    ↓
يرى قائمة الأوامر
    ↓
يختار أمر
    ↓
يحصل على النتيجة
```

---

## 🎓 أمثلة سريعة

### مثال 1: طالب يريد تحليل
```
/analyze
[أرسل: 12345]
→ تحليل مفصل لأدائك
```

### مثال 2: طالب يريد خطة
```
/plan
[أرسل: محمد أحمد]
→ خطة دراسة مخصصة
```

### مثال 3: سؤال عام
```
/ask
[أرسل: ما هي المشتقة؟]
→ شرح ذكي مع أمثلة
```

### مثال 4: مشرف يريد إحصائيات
```
/admin
[اختر: إحصائيات النظام]
→ معلومات كاملة
```
