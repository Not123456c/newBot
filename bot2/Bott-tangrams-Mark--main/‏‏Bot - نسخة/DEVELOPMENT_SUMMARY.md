## 💻 التفاصيل التقنية المتقدمة

### خوارزمية حساب المعدل:

```python
# 1. جلب البيانات
response = supabase.table("all_marks").select("*").execute()
df = pd.DataFrame(response.data)

# 2. تجميع حسب الطالب وحساب المعدل
student_stats = df.groupby(["student_id", "student_name"]).agg(
    average_grade=("total_grade", "mean"),  # متوسط الدرجات
    total_subjects=("total_grade", "count")  # عدد المواد
)

# 3. ترتيب تنازلي
top_n = student_stats.sort_values(by="average_grade", ascending=False).head(n)

# 4. التنسيق والعرض
for i, (_, row) in enumerate(top_n.iterrows()):
    message += f"المركز {i+1}: {row['student_name']} - {row['average_grade']:.2f}"
```
