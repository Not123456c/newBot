# 📖 دليل الأوامر الكامل للبوت

**تاريخ التحديث:** 2026-03-20  
**الحالة:** ✅ تم إضافة جميع الأوامر لـ /help

---

## 🔷 الأوامر العامة (للجميع)

### `/start`
- **الوصف:** البدء والقائمة الرئيسية
- **الاستخدام:** ابدأ الحوار مع البوت
- **الرد:** قائمة الخيارات الرئيسية

### `/top`
- **الوصف:** أفضل 3 طلاب
- **الاستخدام:** اعرض قائمة أفضل 3 طلاب
- **الرد:** صورة وتفاصيل المتفوقين

### `/help`
- **الوصف:** دليل الاستخدام الكامل
- **الاستخدام:** احصل على قائمة جميع الأوامر
- **الرد:** دليل شامل مع شرح جميع الأوامر

---

## 🤖 الأوامر الذكية (AI)

### `/analyze`
- **الوصف:** تحليل أسباب الضعف
- **الاستخدام:** `/analyze` ثم أرسل رقمك الجامعي
- **الميزات:**
  - تحليل ذكي لنقاط الضعف
  - توصيات لتحسين الأداء
  - معلومات عن المواد الضعيفة
- **المثال:**
  ```
  /analyze
  [أرسل: 12345]
  → تحليل ذكي لأسباب ضعفك
  ```

### `/plan`
- **الوصف:** خطة دراسة ذكية وشخصية
- **الاستخدام:** `/plan` ثم أرسل رقمك الجامعي
- **الميزات:**
  - خطة مذاكرة مخصصة
  - معتمدة على نقاط ضعفك
  - توقيتات وأولويات
- **المثال:**
  ```
  /plan
  [أرسل: أحمد محمد]
  → خطة دراسة مفصلة
  ```

### `/ask`
- **الوصف:** أسئلة وإجابات ذكية
- **الاستخدام:** `/ask` ثم أرسل سؤالك
- **الميزات:**
  - إجابات ذكية من Gemini AI
  - لأي سؤال أكاديمي
  - شروحات مفصلة
- **المثال:**
  ```
  /ask
  [أرسل: كيف أحسب المشتقة؟]
  → شرح مفصل مع أمثلة
  ```

## 👨‍💼 أوامر المشرفين فقط ⛔️

### `/admin`
- **الوصف:** لوحة تحكم الإدارة الشاملة
- **الاستخدام:** `/admin`
- **الخيارات المتاحة:**
  
  **1️⃣ إحصائيات النظام 📊**
  - معلومات عن أداء البوت
  - عدد الطلبات
  - استهلاك الموارد

  **2️⃣ عدد المستخدمين 👥**
  - إجمالي عدد المستخدمين
  - النشطين
  - المحظورين

  **3️⃣ تحليلات الدفعة 📈**
  - أداء الدفعة الكاملة
  - المعدل العام
  - توزيع الدرجات

  **4️⃣ سجل الأخطاء 📝**
  - سجل بجميع الأخطاء
  - التفاصيل والحلول
  - معدل الأخطاء

  **5️⃣ استخراج Excel 📥**
  - تصدير جميع البيانات
  - أنماط Excel
  - قابلة للتحليل

  **6️⃣ إرسال تعميم 📢**
  - رسالة لجميع المستخدمين
  - إعلانات مهمة
  - تحديثات

  **7️⃣ الـ 10 الأوائل 🏆**
  - أفضل 10 طلاب
  - بناءً على المعدل
  - مع التفاصيل

  **8️⃣ الأوائل بالمادة 📚**
  - أفضل طالب بكل مادة
  - تصنيف متكامل
  - مقارنات

  **9️⃣ إدارة المحظورين 🚫**
  - قائمة المحظورين
  - إضافة/حذف
  - الأسباب

  **🔟 حوادث مزعجة معلقة 🚨**
  - الرسائل المزعجة
  - الإجراءات المعلقة
  - المعالجة

### `/backup`
- **الوصف:** نسخ احتياطي يدوي فوري
- **الاستخدام:** `/backup`
- **النتيجة:**
  - ✅ تم النسخ الاحتياطي بنجاح
  - عرض الملفات المنسوخة
  - ملفات الخطأ (إن وجدت)
- **المثال:**
  ```
  /backup
  → ⏳ جاري النسخ...
  → ✅ تم النسخ (10 ملفات)
  ```

### `/backup_status`
- **الوصف:** حالة النسخ الاحتياطي الأخيرة
- **الاستخدام:** `/backup_status`
- **المعلومات:**
  - الحالة (🟢 نشط / 🔴 متوقف)
  - آخر نسخة
  - التاريخ والوقت
  - عدد الملفات
- **المثال:**
  ```
  /backup_status
  → 🟢 نشط
  → آخر نسخة: 2026-03-20 10:30
  → 15 ملف
  ```

### `/upload`
- **الوصف:** رفع ملفات (صور، مستندات)
- **الاستخدام:** `/upload` ثم ارسل الملف
- **الملفات المدعومة:**
  - صور (JPG, PNG, GIF)
  - مستندات (PDF, DOC)
  - DATA (JSON, Excel)
- **الحدود:**
  - أقصى حجم: 50 MB
  - أقصى عدد: 10 ملفات
- **المثال:**
  ```
  /upload
  [أرسل صورة]
  → ✅ تم رفع الملف بنجاح
  ```
